# chatter

Syncs AI skill files to a shared team repo via PR. One person's agent learns something, the whole team gets it on next pull.

```
skill file -> chatter opens PR -> team reviews -> everyone pulls
```

## How it works

1. You have a skill file (written by hand, generated by your agent, extracted from a session, however you got it)
2. `chatter sync` copies it to the shared team repo and opens a PR
3. Team reviews and merges
4. Everyone pulls. Skills land in both opencode and Claude Code directories.

Skills are namespaced by project (`my-project:some-fix.md`, `general:some-pattern.md`), auto-detected from your current git remote. If a skill with the same name already exists, chatter updates it instead of creating a duplicate.

## Setup

### 1. Create a shared team repo

Use the template in `team-repo-template/` or make one from scratch:

```bash
gh repo create your-org/team-skills --private
```

### 2. Install on each machine

```bash
curl -fsSL https://raw.githubusercontent.com/rhighs/chatter/main/install.sh | bash -s -- https://github.com/YOUR-ORG/team-skills
```

This clones the team repo, symlinks all skills into place via `stow` (opencode and Claude Code), and writes config to `~/.config/chatter/team.conf`.

### 3. Sync a skill

```bash
chatter sync path/to/my-skill.md
```

Any `.md` file works.

## Trigger via AI

Tell your agent "sync this skill" or "share this to the team" and it calls `chatter sync <file>` directly.

## Pull latest

```bash
chatter pull
```

## Namespace convention

`{project}:{description}.md`, derived from `git remote get-url origin`.

```
my-project:mcp-proxy-namespaced-params.md
general:git-rebase-conflict-resolution.md
pleasetriage:vercel-dns-config.md
```

## Team repo structure

```
team-skills/
  opencode/              stowed to ~/.config/opencode/commands/
    my-project:fix.md
  claude/                stowed to ~/.claude/skills/
    my-project:fix/
      SKILL.md
  README.md
```

## Requirements

- `git`, `gh` (GitHub CLI)
- `stow` (`brew install stow` / `apt install stow`)
